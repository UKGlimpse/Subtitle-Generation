How it works

  • Your browser captures the video's audio track using captureStream().
  • The audio is recorded as a WebM file in-memory.
  • The file is sent to OpenAI's transcription endpoint with response_format=vtt.
  • The resulting WebVTT file is attached back to the video and is ready to download.

Built for GitHub Pages. For production use, proxy the API requests through your own backend to keep keys secure.
